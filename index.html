一周爱心菜谱

每天为你做顿饭，是我想表达的小浪漫💖

周一：辣椒炒肉 + 清炒空心菜
主料 & 配菜：
猪里脊肉/五花肉 200g
青椒（或红椒）2个（约150g）
空心菜 300g
大蒜 3瓣 / 生姜 2片
调料类：
生抽 1.5汤匙 / 老抽 半汤匙
料酒 1汤匙 / 盐 适量 / 白糖 少许 / 食用油 适量
做法：
1. 猪肉洗净切薄片，加入生抽、料酒腌制10分钟。
2. 青椒去籽切条，大蒜剁末。
3. 热锅凉油，放入姜片、蒜末爆香。
4. 放入肉片中火翻炒至变色后盛出备用。
5. 锅中留底油，放入青椒快炒1分钟，加少许水焖半分钟。
6. 倒回肉片一起翻炒均匀。
7. 加老抽调色，加盐、少许白糖提鲜，翻炒几下即可出锅。
周二：豆芽炒肉 + 蒜蓉西兰花
主料 & 配菜：
猪瘦肉 200g
黄豆芽 200g / 胡萝卜丝 50g（可选）
西兰花 1颗（约300g）
大蒜 2瓣
调料类：
生抽 1.5汤匙 / 料酒 1汤匙 / 淀粉 1茶匙
盐 适量 / 食用油 适量 / 蚝油 1汤匙
做法：
1. 瘦肉切丝，加入生抽、料酒、淀粉拌匀腌制10分钟。
2. 黄豆芽洗净，胡萝卜切细丝。
3. 热锅加油，先炒肉丝至变色盛出备用。
4. 另起锅放油，爆香蒜末，加入豆芽和胡萝卜丝快炒。
5. 倒入炒好的肉丝翻炒均匀。
6. 西兰花切成小朵焯水1分钟后与蒜末快炒，加蚝油调味。
周三：酸汤肥牛 + 酸辣土豆丝
主料 & 配菜：
肥牛卷 300g
金针菇 100g / 豆芽 100g / 土豆 2个（约300g）
泡椒/野山椒 5根 / 干辣椒 5个 / 蒜末 3瓣 / 葱花 适量
姜片 3片
调料类：
番茄酱 2汤匙 / 盐 适量 / 白胡椒粉 少许 / 鸡精 少许
醋 1汤匙 / 白糖 少许
做法：
1. 所有食材准备好：泡椒剪段，金针菇撕开洗净，豆芽洗净。
2. 锅中加水烧开，放入姜片、泡椒煮3分钟出味。
3. 加入番茄酱搅拌均匀，继续煮2分钟。
4. 先放入金针菇煮1分钟，再放豆芽煮1分钟。
5. 分次放入肥牛卷，烫熟即可（不宜久煮）。
6. 土豆去皮切丝，清水冲洗掉淀粉，控干水分。
7. 干辣椒剪段，蒜剁末，热锅加油爆香。
8. 放入土豆丝大火快炒1~2分钟，加醋、盐、糖调味即可。
周四：白灼虾 + 辛辣鸡蛋炒青椒
主料 & 配菜：
鲜虾 300g / 鸡蛋 3个 / 青椒 2个（约150g）
姜片 3片 / 葱段 2根 / 蒜末 2瓣 / 葱花 少许
调料类：
料酒 1汤匙 / 生抽 适量 / 醋 适量 / 盐 适量 / 食用油 适量
做法：
1. 虾剪须、去虾线，清洗干净。
2. 锅中加水，放入姜片、葱段、料酒，大火烧开。
3. 放入虾煮约2分钟，虾身变红立即捞出。
4. 搭配蘸料：蒜末+生抽+醋调汁，即可食用。
5. 鸡蛋打散，加少许盐。
6. 青椒洗净切丝。
7. 热锅加油，倒入鸡蛋液炒至半凝固状态盛出。
8. 再起锅加油，放入青椒炒至断生，倒入鸡蛋翻炒均匀，加盐调味，撒葱花即可。
周五：回锅肉 + 西红柿炒鸡蛋
主料 & 配菜：
带皮五花肉 300g / 青蒜苗 100g
鸡蛋 3个 / 西红柿 2个（约250g）
姜片 3片 / 葱段 1根 / 葱花 少许
调料类：
郫县豆瓣酱 1.5汤匙 / 甜面酱 0.5汤匙
料酒 1汤匙 / 糖 少许 / 盐 适量 / 食用油 适量
做法：
1. 五花肉冷水下锅，加姜、葱、料酒煮至七八成熟，捞出晾凉切片。
2. 热锅不放油，放入肉片煸炒出油脂。
3. 加入姜片、豆瓣酱炒香。
4. 加甜面酱、糖炒匀。
5. 加入青蒜苗翻炒至断生即可。
6. 鸡蛋加盐打散，热锅加油炒至凝固盛出。
7. 西红柿切块，热锅加油炒出沙。
8. 倒入鸡蛋一起翻炒，加盐、少许糖调味，撒葱花即可。
周六：辣子鸡丁 + 手撕包菜
主料 & 配菜：
鸡胸肉/鸡腿肉 250g / 包菜 半个（约300g）
干辣椒 8个 / 花椒 10粒 / 葱姜蒜 适量
干辣椒 3个 / 蒜末 3瓣 调料类：
生抽 1.5汤匙 / 料酒 1汤匙 / 淀粉 1茶匙 / 盐 适量 / 糖 少许 / 食用油 适量
做法：
1. 鸡肉切丁，加生抽、料酒、淀粉腌制10分钟。
2. 热锅加油，放入鸡丁炸至表面微黄捞出。
3. 锅中留底油，放入干辣椒、花椒、葱姜蒜炒香。
4. 加入鸡丁翻炒，调入盐、少许糖即可。
5. 包菜手撕成小块，洗净沥干。
6. 干辣椒剪段，蒜剁末。
7. 热锅加油，先爆香干辣椒和蒜末。
8. 放入包菜大火翻炒1~2分钟。
9. 加盐、生抽、少许醋调味，炒匀即可。
周日：口蘑汤 + 凉拌手撕鸡
主料 & 配菜：
口蘑 10朵（约200g） / 鸡蛋 1个（可选）
鸡胸肉/鸡腿肉 200g / 黄瓜丝 50g / 香菜 适量
蒜末 3瓣 / 小米辣 1个 / 花生碎 适量
葱花 适量 / 清水 500ml 调料类：
生抽 1.5汤匙 / 醋 1汤匙 / 香油 半汤匙 / 辣椒油 1汤匙 / 糖 少许 / 盐 适量 / 鸡精 少许
做法：
1. 口蘑洗净切片，锅中加清水煮沸，放入口蘑煮3分钟。
2. 打入鸡蛋搅匀（可选），加盐、鸡精调味，撒葱花，淋几滴香油即可。
3. 鸡肉冷水下锅，加姜片、料酒煮熟，放凉后撕成丝。
4. 黄瓜切丝，香菜切段。
5. 准备调料汁：蒜末、小米辣、生抽、醋、糖、香油、辣椒油混合。
6. 鸡丝与黄瓜丝混合，淋上调料汁，撒花生碎和香菜即可。
